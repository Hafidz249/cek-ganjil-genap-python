## 🧐 Deskripsi Proyek
Proyek ini dibuat untuk mendemonstrasikan pemahaman dasar mengenai:
1.  **Operator Modulo (`%`):** Digunakan untuk mengecek sisa pembagian bilangan dengan angka 2.
2.  **Exception Handling:** Implementasi `try-except` untuk mencegah aplikasi *crash* akibat input non-numerik.
3.  **Clean Code:** Penulisan fungsi yang modular dan mudah dibaca.
